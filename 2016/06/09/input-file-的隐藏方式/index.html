input file 的隐藏方式

在需要上传文件的过程中， 需要 <input type="file"> 控件，但往往需要对其进行美化。

<button>上传文件</button>
<input type="file">

在这种结构下，设置input css:

input[type="file"]{ 
    position: absolute;
    opacity: 0;
    top: 50px;
    left:50px;
    cursor:pointer;
}

这种情况下，hover效果下， cursor:pointer失效， 还是input file 默认的样式，为了解决这个，可以利用属性 font-size:200px 来实现。

input[type="file"]{ 
    position: absolute;
    opacity: 0;
    top: 50px;
    left:50px;
    font-size:200px;
    cursor:pointer;
}

这样 cursor:pointer 可以了， 但还有个问题，font-size:200px 使控件的区域变得很大，鼠标点在button外部也能触发file。

<div class="container">
    <button>上传文件</button>
    <input type="file" />
</div>

这种结构下，设置 .container{overflow:hidden} 解决了问题。
